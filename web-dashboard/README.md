# Web Dashboard (Planned)

Recommended stack:
- Next.js + TypeScript
- Tailwind CSS + component library (e.g., shadcn/ui)
- TanStack Query for API state
- Recharts/ECharts for analytics visualizations

Core pages:
- `/products`
- `/orders`
- `/stores`
- `/automation`
- `/analytics`
- `/settings`

The dashboard should consume backend endpoints under `/api/*` and use JWT auth.
